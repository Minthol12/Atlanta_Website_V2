About the Atlanta NewsCast Project

Vision and Purpose

The Atlanta NewsCast project was developed to modernize the school's digital presence, offering students, staff, and the community a centralized, secure, and user-friendly portal for all critical information. Key features include a personalized Student Hub, a modular Admin Dashboard, a comprehensive How-To section, and a full-scale news and sports platform. The design prioritizes accessibility, mobile responsiveness, and a professional dark-mode aesthetic.

Technical Stack Highlights:

  • Front-End: Modern HTML5, Modular CSS (using variable themes for Dark/Light mode).
  • Interactivity: Vanilla JavaScript for dynamic elements (Countdown Timer).
  • Security: Implementation of strong security headers and CSP Nonce to defend against XSS attacks.
